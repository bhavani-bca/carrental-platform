# Car Rental Platform

## Introduction
This is a full-stack Car Rental Platform built using Java, Spring Boot, Spring Security, Spring Data JPA, Thymeleaf, HTML, CSS, and JavaScript.

## Features
- User authentication with 2FA (email OTP)
- Car management and inventory
- Booking system
- Responsive UI
- Admin panel

## Technologies Used

### Backend (Java + Spring)
- Java
- Spring Boot
- Spring Security
- Spring Data JPA
- H2 / MySQL

### Frontend (HTML / CSS / JS / Thymeleaf)
- HTML5
- CSS3
- JavaScript
- Bootstrap
- Thymeleaf

### Tools
- Maven
- IntelliJ IDEA / Eclipse
- GitHub

## System Architecture
Frontend → Spring Boot Backend → H2/MySQL  
Backend uses MVC architecture with Controller, Service, Repository layers.
